Cyber Secure Encryption Suite

A Python-based Cybersecurity Toolkit that demonstrates both classical and modern encryption techniques with additional security features.

## 📌 Project Overview

Cyber Secure Encryption Suite is a Python application developed to help users encrypt and decrypt data securely. It combines the Caesar Cipher algorithm with AES encryption and includes extra cybersecurity utilities such as password strength checking, file encryption, encryption history, and security tips.

---

## ✨ Features

- 🔑 Caesar Cipher Encryption & Decryption
- 🔐 AES Encryption & Decryption
- 💪 Password Strength Checker
- 📂 File Encryption & Decryption
- 📝 Encryption History Logging
- 🔒 Automatic AES Key Generation
- 💡 Random Cybersecurity Tips
- 🖥️ User-Friendly Menu Interface
