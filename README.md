\# Caesar Cipher \& AES Encryption Tool



\## Objective

This project demonstrates two encryption techniques:

\- Caesar Cipher

\- AES Encryption (using Python Cryptography Library)



\## Features

\- Caesar Cipher Encryption

\- Caesar Cipher Decryption

\- AES Encryption

\- AES Decryption

\- Automatically Generates Encryption Key



\## Technologies Used

\- Python 3

\- Cryptography Library

\- Visual Studio Code



\## How to Run

1\. Install Python

2\. Install cryptography:

&#x20;  pip install cryptography

3\. Run:

&#x20;  python main.py
